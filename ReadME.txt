Constraints

1. Run the following command to install all the dependencies mentioned in the requirements file
	 pip -r install requirements.txt
2. To run the whole code use "make run"
3. The link attached below has all the complete project with the datasets and models required.
	Link: https://drive.google.com/drive/folders/1--V-uzO4X7V4TJ-FSbVILJVZgc0M3p_v?usp=sharing
		

Folder Structure
--------------------------------
1. Main folder name is IR on Bhagavad Gita 
2. Main folder has 5 sub folders. One for webscrapping data for all 4 languages and 4 language folders.
3. Webscrapping folder has data in the format of CSV for all languages.
4. Each Language folder is having other folders containing the models implemented.
5. To check the output please find the ipynb file with the word "util".
6. Run the util file by giving the query in the variable with the same name "query" inside the file.
